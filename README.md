# 📉 Customer Churn Prediction & CRM Analytics Dashboard

A machine learning web application that predicts customer churn using Logistic Regression, built with Python and Streamlit.

## 🛠️ Tech Stack
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Streamlit

## 📁 Dataset
- 8,000 customer records with 19 features
- Churn Rate: 48.29%

## 🤖 Model
Trained and compared 5 models — Logistic Regression, Decision Tree, Random Forest, KNN, and SVM. **Logistic Regression** was selected as the final model based on highest accuracy.

## 🖥️ Dashboard
Interactive Streamlit app with dataset upload, visualizations, churn predictions, customer segmentation, revenue analytics, and business insights.

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
